### 配置文件：

* [Homepage_compose.yaml](https://github.com/BroccoliYn618/Homepage/blob/main/Homepage_compose.yaml)为homepage的compose部署文件。
* [setting_broccoli.yaml](https://github.com/BroccoliYn618/Homepage/blob/main/settings_broccoli.yaml)为homepage的设置文件。
* [widgets_broccoli.yaml](https://github.com/BroccoliYn618/Homepage/blob/main/widgets_broccoli.yaml)为homepage的顶部导航文件。
* [services_broccoli.yaml](https://github.com/BroccoliYn618/Homepage/blob/main/services_broccoli.yaml)为homepage的中心展示文件。
* [bookmarks_broccoli.yaml](https://github.com/BroccoliYn618/Homepage/blob/main/bookmarks_broccoli.yaml)为homepage的底部书签文件。

<br>

### 食用方法：

1. Docker通过compose创建homepage容器，注意改为自己的地址映射
2. 选择需要的配置文件
3. 将配置文件内部的示例地址改为自己的地址（example.com）
4. 放到对映的homepage_config文件夹
5. 重启容器开始食用
